关于 ToolBox

我们相信：好工具应该免费，你的数据应该只属于你。

我们的使命

互联网上充斥着形形色色的在线工具，但大多数都有一个共同的问题：它们需要你上传文件到服务器。这意味着你的简历、身份证照片、私密文档都会经过服务人（甚至可能被存储、分析和泄露）。

ToolBox 的诞生就是为了改变这一点。我们利用现代浏览器的强大能力（WebAssembly、Canvas API、Web Workers），让所有处理都在你的设备上完成——你的文件从未离开你的电脑。

我们的承诺

零数据上传

所有文件处理 100% 在浏览器本地运行。我们没有后端服务器，也不收集任何文件内容。你甚至可以断网使用。

完全透明

ToolBox 的全部代码在 GitHub 开源。任何人都可以审查我们的代码，确认没有任何可疑的数据收集行为。

永久免费

核心工具永远免费。不限制文件大小，不限制使用次数。我们通过广告和可选的高级功能来维持运营。

技术架构

ToolBox 完全基于前端技术构建，没有后端服务器参与文件处理：

HTML5 + CSS3 + JavaScript
WebAssembly (PDF 处理)
Canvas API (图片处理)
Web Workers (后台运算)
Vercel 静态托管
GitHub 开源
