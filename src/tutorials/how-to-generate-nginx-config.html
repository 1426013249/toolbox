📋 操作步骤

1

设置基本信息

打开 Nginx 配置生成器，填写域名和监听端口。

2

开启需要的功能

按需配置：

  • 反向代理— 设置后端服务地址
  • SSL / HTTPS— 配置证书路径
  • 静态文件— 设置根目录和索引文件
  • 负载均衡— 添加多个后端节点
3

复制配置到服务器

右侧实时生成 Nginx 配置文件。点击「复制」按钮，粘贴到服务器的 /etc/nginx/conf.d/ 目录下。

💡 小技巧生成配置后记得先用 nginx -t 测试语法，再用 nginx -s reload 重新加载。
