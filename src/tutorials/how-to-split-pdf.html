如何拆分PDF文件？按页码范围自由提取

📖 阅读 2 分钟🔄 更新于 2026-02🔒 100% 本地处理
需要从一个大 PDF 中提取几页？或者把每一页都拆成单独文件？ToolBox 提供两种拆分模式，免费、本地处理、不上传服务器。

🚀 直接开始拆分

不想看教程？直接用工具

打开 PDF 拆分工具 →

📋 操作步骤

1

上传 PDF 文件

打开 PDF 拆分工具，上传或拖拽你的 PDF 文件。

2

选择拆分模式

ToolBox 提供两种拆分方式：

  • 逐页拆分— 每一页保存为单独的 PDF 文件
  • 按范围拆分— 输入页码范围（如 1-3, 5, 8-10），只提取指定页面
3

下载拆分后的文件

点击「拆分」按钮。逐页拆分会打包成 ZIP 下载，按范围拆分直接下载 PDF。文件全程不离开你的浏览器。

🔒 隐私保障所有 PDF 拆分在浏览器本地完成，文件不经过服务器。适合处理合同、证件等敏感文档。
